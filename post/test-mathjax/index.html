By Hugo Authors | May 22, 2021 | 1 minutes

$$ \int_{a}^{b} x^2 dx $$

$$ \varphi = 1+\frac{1} {1+\frac{1} {1+\frac{1} {1+\cdots} } } $$

Enable MathJax in the config file by setting the mathjax param to true. This will import the necessary MathJax CSS/JS.
